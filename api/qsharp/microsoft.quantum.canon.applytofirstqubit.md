Névtér: [Microsoft. Quantum. Canon](xref:Microsoft.Quantum.Canon)

Egy műveletet alkalmaz a regisztráció első qubit.

operation ApplyToFirstQubit (op : (Qubit => Unit), register : Qubit[]) : Unit
