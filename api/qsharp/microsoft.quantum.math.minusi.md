```qsharp
function MinusI (a : Int, b : Int) : Int
```


## <a name="input"></a>Bevitel

### <a name="a--int"></a>a: [int](xref:microsoft.quantum.lang-ref.int)

## <a name="output--int"></a>Kimenet: [int](xref:microsoft.quantum.lang-ref.int)
