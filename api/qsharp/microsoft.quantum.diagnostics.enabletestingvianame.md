# <a name="enabletestingvianame-user-defined-type"></a>EnableTestingViaName-felhasználó által definiált típus

Névtér: [Microsoft. Quantum. Diagnostics](xref:Microsoft.Quantum.Diagnostics)

Csomag: [Microsoft. Quantum. QSharp. Core](https://nuget.org/packages/Microsoft.Quantum.QSharp.Core)


A fordító által felismert attribútum, amelyen keresztül egy alternatív név meghatározható, amelyet a rendszer a típus vagy a meghívásos tesztelési célú betöltéskor használhat.

```qsharp

@ Microsoft.Quantum.Core.Attribute()
newtype EnableTestingViaName = (String);
```
