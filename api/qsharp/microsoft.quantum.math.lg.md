# <a name="lg-function"></a>LG-függvény

Névtér: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Csomag [](https://nuget.org/packages/)


Kiszámítja egy szám Base-2 logaritmusát.

```qsharp
function Lg (input : Double) : Double
```


## <a name="input"></a>Bevitel

### <a name="input--double"></a>bemenet: [dupla](xref:microsoft.quantum.lang-ref.double)

Egy valós szám $x $.



## <a name="output--double"></a>Kimenet: [dupla](xref:microsoft.quantum.lang-ref.double)

A Base-2 logaritmus $y = \ log_2 (x) $, például $x = 2 ^ y $.
