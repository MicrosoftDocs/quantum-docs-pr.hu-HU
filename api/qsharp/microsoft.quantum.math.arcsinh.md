Névtér: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Csomag: [Microsoft. Quantum. Standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Egy szám inverz szinusz hiperbolikusát számítja ki.

```qsharp
function ArcSinh (x : Double) : Double
```


## <a name="input"></a>Bevitel

### <a name="x--double"></a>x: [dupla](xref:microsoft.quantum.lang-ref.double)

## <a name="output--double"></a>Kimenet: [dupla](xref:microsoft.quantum.lang-ref.double)

Egy valós szám $y $, például $x = \operatorname{sinh} (y) $.
