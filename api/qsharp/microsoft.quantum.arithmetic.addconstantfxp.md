# <a name="addconstantfxp-operation"></a>AddConstantFxP művelet

Névtér: [Microsoft. Quantum. aritmetika](xref:Microsoft.Quantum.Arithmetic)

Csomag [](https://nuget.org/packages/)


Egy klasszikus állandót hoz létre egy rögzített számú kvantum-ponthoz.

```qsharp
operation AddConstantFxP (constant : Double, fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Unit
```


## <a name="input"></a>Bevitel

### <a name="constant--double"></a>állandó: [dupla](xref:microsoft.quantum.lang-ref.double)

A kvantum rögzített pont számához hozzáadandó állandó.


### <a name="fp--fixedpoint"></a>FP: [FixedPoint](xref:Microsoft.Quantum.Arithmetic.FixedPoint)

Az a rögzített pontú szám, amelyhez a konstans hozzá lesz adva.
