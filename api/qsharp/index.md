# <a name="q-libraries"></a>Q#-kódtárak #

A szabványos Q#-kódtár olyan műveleteket, függvényeket és felhasználó által definiált típusokat tartalmaz, amelyek kvantumprogramok fejlesztéséhez használhatóak. Emellett a szabványos kódtárra épülő három további kódtár is elérhető – a kémiai, numerikus és gépi tanulási kódtár.

Tovább információk a Q#-kódtárakról: <xref:microsoft.quantum.libraries>.
