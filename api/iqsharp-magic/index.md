# <a name="iq-magic-commands"></a>IQ# Magic parancsok
| Magic parancs | Összefoglalás |
|---------------|---------|
| [`%azure.connect`](xref:microsoft.quantum.iqsharp.magic-ref.azure.connect) | Kapcsolódás az Azure Quantum-munkaterülethez vagy a kapcsolat jelenlegi állapotának megjelenítése. |
| [`%azure.execute`](xref:microsoft.quantum.iqsharp.magic-ref.azure.execute) | Feladat végrehajtása az Azure Quantum-munkaterületen. |
| [`%azure.jobs`](xref:microsoft.quantum.iqsharp.magic-ref.azure.jobs) | Megjeleníti az aktuális Azure Quantum-munkaterületen lévő feladatok listáját. |
| [`%azure.output`](xref:microsoft.quantum.iqsharp.magic-ref.azure.output) | Megjeleníti az aktuális Azure Quantum-munkaterületen lévő egyik feladat eredményét. |
| [`%azure.status`](xref:microsoft.quantum.iqsharp.magic-ref.azure.status) | Megjeleníti az aktuális Azure Quantum-munkaterületen lévő egyik feladat állapotát. |
| [`%azure.submit`](xref:microsoft.quantum.iqsharp.magic-ref.azure.submit) | Feladat elküldése egy Azure Quantum-munkaterületre. |
| [`%azure.target`](xref:microsoft.quantum.iqsharp.magic-ref.azure.target) | Beállítja vagy megjeleníti egy Q#-feladat beküldésének aktív végrehajtási célját egy Azure Quantum-munkaterületen. |
| [`%check_kata`](xref:microsoft.quantum.iqsharp.magic-ref.check_kata) | Ellenőrzi a referenciaimplementációt egyetlen Kata-teszt esetében. |
| [`%chemistry.broombridge`](xref:microsoft.quantum.iqsharp.magic-ref.chemistry.broombridge) | Betölti és visszaadja egy Broombridge elektronikus szerkezeti probléma megjelenítését egy adott .yaml fájlból. |
| [`%chemistry.encode`](xref:microsoft.quantum.iqsharp.magic-ref.chemistry.encode) | A fermion Hamilton-operátorokat a Q# által használható formátumban kódolja. |
| [`%chemistry.fh.add_terms`](xref:microsoft.quantum.iqsharp.magic-ref.chemistry.fh.add_terms) | Kifejezéseket ad hozzá a fermion Hamilton-operátorokhoz. |
| [`%chemistry.fh.load`](xref:microsoft.quantum.iqsharp.magic-ref.chemistry.fh.load) | Betölt egy fermion Hamilton-operátort egy elektronikus szerkezeti problémához. A probléma betöltése egy fájlból történik, vagy argumentumként van megadva. |
| [`%chemistry.inputstate.load`](xref:microsoft.quantum.iqsharp.magic-ref.chemistry.inputstate.load) | Betölti egy Broombridge elektronikus szerkezeti probléma megjelenítését, és visszaadja a választott bemeneti állapotot. |
| [`%config`](xref:microsoft.quantum.iqsharp.magic-ref.config) | Engedélyezi a beállítási vagy lekérdezési konfigurációs lehetőségeket. |
| [`%estimate`](xref:microsoft.quantum.iqsharp.magic-ref.estimate) | Futtat egy adott függvényt vagy műveletet a ResourcesEstimator célszámítógépen. |
| [`%kata`](xref:microsoft.quantum.iqsharp.magic-ref.kata) | Egyetlen tesztet hajt végre. |
| [`%lsmagic`](xref:microsoft.quantum.iqsharp.magic-ref.lsmagic) | Visszaadja a jelenleg elérhető magic parancsok listáját. |
| [`%lsopen`](xref:microsoft.quantum.iqsharp.magic-ref.lsopen) | Az aktuálisan megnyitott névtereket és azok aliasait sorolja fel. |
| [`%package`](xref:microsoft.quantum.iqsharp.magic-ref.package) | Lehetővé teszi NuGet-csomagok betöltését. |
| [`%performance`](xref:microsoft.quantum.iqsharp.magic-ref.performance) | Jelenti a kernel aktuális teljesítménymetrikáit. |
| [`%project`](xref:microsoft.quantum.iqsharp.magic-ref.project) | A Q#-projekthivatkozások megtekintésére vagy hozzáadására szolgáló képességet biztosítja. |
| [`%simulate`](xref:microsoft.quantum.iqsharp.magic-ref.simulate) | Futtat egy adott függvényt vagy műveletet a QuantumSimulator célszámítógépen. |
| [`%toffoli`](xref:microsoft.quantum.iqsharp.magic-ref.toffoli) | Futtat egy adott függvényt vagy műveletet a ToffoliSimulator célszámítógépen. |
| [`%trace`](xref:microsoft.quantum.iqsharp.magic-ref.trace) | A megadott művelet végrehajtási útvonalának vizualizációja. |
| [`%who`](xref:microsoft.quantum.iqsharp.magic-ref.who) | Listázza az aktuális munkamenetben elérhető Q#-műveleteket. |
| [`%workspace`](xref:microsoft.quantum.iqsharp.magic-ref.workspace) | Az aktuális munkaterülethez kapcsolódó műveleteket biztosít. |
