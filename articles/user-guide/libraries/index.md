- <span data-ttu-id="9a79d-108">[**A kvantumnumerikus kódtár**](xref:microsoft.quantum.numerics.intro): Ez a szakasz ismerteti a kvantumnumerikus kódtárat, amely számos matematikai függvény implementációját biztosítja.</span><span class="sxs-lookup"><span data-stu-id="9a79d-108">[**Quantum numerics library**](xref:microsoft.quantum.numerics.intro): This section describes the quantum numerics library, which provides implementations for a host of mathematical functions.</span></span> <span data-ttu-id="9a79d-109">A kódtár támogatja az egész számos (előjeles és előjel nélküli) és a fixpontos jelöléseket.</span><span class="sxs-lookup"><span data-stu-id="9a79d-109">It supports integer (signed & unsigned) and fixed-point representations.</span></span>
