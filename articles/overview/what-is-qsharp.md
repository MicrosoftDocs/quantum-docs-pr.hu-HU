# <a name="what-is-q"></a><span data-ttu-id="d296f-103">Mi az a Q#?</span><span class="sxs-lookup"><span data-stu-id="d296f-103">What is Q#?</span></span>

<span data-ttu-id="d296f-104">A Q# egy jellemzően a kvantum-számítástechnikában használható programozási nyelv.</span><span class="sxs-lookup"><span data-stu-id="d296f-104">Q# is a programming language with features that are special to quantum computing.</span></span> <span data-ttu-id="d296f-105">A Q# egy olyan keretrendszert kínál a kvantumprogramozóknak, amely segít az algoritmusokra koncentrálni anélkül, hogy olyan technikai részletekkel kéne törődni, mint a kapuszekvencia optimalizálása vagy egy kvantumszámítógép fizikai implementálása.</span><span class="sxs-lookup"><span data-stu-id="d296f-105">Q# provides quantum programmers a framework that allows you to focus on the algorithms without having to care about technical details like gate sequence optimization or the physical implementation of a quantum computer.</span></span>

## <a name="code-algorithms"></a><span data-ttu-id="d296f-107">Kódalgoritmusok</span><span class="sxs-lookup"><span data-stu-id="d296f-107">Code algorithms</span></span>

<span data-ttu-id="d296f-108">A kvantum-számítástechnika korai időszakában az algoritmusokat még diagramként vizualizálták, hasonlóan a klasszikus számítástechnikában használt kapcsolási rajzokhoz.</span><span class="sxs-lookup"><span data-stu-id="d296f-108">In the early days of quantum computing algorithms were visualized as diagrams similarly to circuit diagrams in classical computing.</span></span>  <span data-ttu-id="d296f-109">Bár a körmodell sok éven át hasznosnak bizonyult a kvantum-számítástechnikai kutatásokban, a Microsoftnál úgy gondoljuk, hogy a fejlesztők túl tudnak lépni a kvantumkörökön, és a Q# segítségével arra is képesek lehetnek, hogy kvantumalgoritmusokat és -alkalmazásokat fejlesszenek ki.</span><span class="sxs-lookup"><span data-stu-id="d296f-109">While the circuit model has been very useful for many years in quantum computing research, here at Microsoft, we believe that developers can go beyond quantum circuits and develop quantum algorithms and applications using Q#.</span></span> <span data-ttu-id="d296f-110">A Q# úgy lett kialakítva, hogy építkezzen a klasszikus szoftverfejlesztésből évtizedek alatt leszűrt tapasztalatokra, és magas szintű programnyelvi funkciókat adjon a kvantumfejlesztők kezébe, amelyek kifejezetten a kvantum-számítástechnikára vannak kiélezve.</span><span class="sxs-lookup"><span data-stu-id="d296f-110">The Q# language was built to take advantage of what we’ve learned through decades of classical software development, and empower quantum developers with high-level language functionality specifically targeted for quantum computing.</span></span>


## <a name="how-does-q-work"></a><span data-ttu-id="d296f-111">Hogyan működik a Q#?</span><span class="sxs-lookup"><span data-stu-id="d296f-111">How does Q# work?</span></span>

<span data-ttu-id="d296f-112">A Q# egyik alapvető építőeleme a `Qubit` típus, amit nem lehet lemásolni vagy közvetlenül elérni, pontosan úgy, mint egy igazi qubitet.</span><span class="sxs-lookup"><span data-stu-id="d296f-112">One of the fundamental building blocks of Q# is the `Qubit` type, which cannot be copied or directly accessed, just like a real qubit.</span></span> <span data-ttu-id="d296f-113">Ehelyett megmérhetjük, az eredményt pedig egy `Result` változóban tárolhatjuk, ami egy Q#-típus két lehetséges értékkel: `Zero` és `One`.</span><span class="sxs-lookup"><span data-stu-id="d296f-113">Instead, we can measure it and store the outcome of the measurement in a `Result` variable, a Q# type that can take two possible values: `Zero` and `One`.</span></span> <span data-ttu-id="d296f-114">Az ehhez hasonló konstrukciók garantálják, hogy az algoritmusok mindig tiszteletben tartják a kvantumfizika törvényeit, és megfelelően futnak a kvantumszámítógépeken és szimulátorokon.</span><span class="sxs-lookup"><span data-stu-id="d296f-114">Constructs like this one guarantee that algorithms always respect the laws of quantum physics and can run correctly on quantum computers or simulators.</span></span>

<span data-ttu-id="d296f-115">A Q# emellett olyan klasszikus logikai függvényeket is tartalmaz, mint a feltételes értékek vagy a hurkok, néhány apró részlettel egyetemben, amelyek gondoskodnak az összes kvantumszabály betartásáról.</span><span class="sxs-lookup"><span data-stu-id="d296f-115">Q# also includes classical logic features like conditionals or loops with some subtleties to make sure that all the quantum rules are being respected.</span></span> <span data-ttu-id="d296f-116">Például a kvantumműveleteknek visszafejthetőnek kell lenniük.</span><span class="sxs-lookup"><span data-stu-id="d296f-116">For example, quantum operations need to be reversible.</span></span> <span data-ttu-id="d296f-117">Ez némi korlátozást jelent a hurkok végrehajtása terén.</span><span class="sxs-lookup"><span data-stu-id="d296f-117">This enforces some constraints on the way loops are executed.</span></span>

<span data-ttu-id="d296f-118">A Q#-programok gyakran vannak párban egy C# vagy Python nyelven írt gazdaprogrammal, amely biztosítja a klasszikus és a kvantumkód megfelelő szervezettségét.</span><span class="sxs-lookup"><span data-stu-id="d296f-118">Q# programs are often paired with a host program written in C# or Python, which can provide convenient organization of classical and quantum code.</span></span> <span data-ttu-id="d296f-119">A .NET-nyelvek, például a C# és a Python támogatása mellett a QDK Jupyter-notebook-támogatást is nyújt az IQ# Jupyter kernel révén.</span><span class="sxs-lookup"><span data-stu-id="d296f-119">In addition to supporting .NET languages such as C# and Python, the QDK provides Jupyter Notebook support with the IQ# Jupyter kernel.</span></span>

## <a name="use-q-to-learn-quantum-computing"></a><span data-ttu-id="d296f-120">A Q# használata a kvantum-számítástechnika elsajátításához</span><span class="sxs-lookup"><span data-stu-id="d296f-120">Use Q# to learn quantum computing</span></span>

<span data-ttu-id="d296f-121">Mostanáig a kvantum-számítástechnika elsajátításához meg kellett tanulni a kapcsolatcsoport-modellt, hogy az algoritmusokat a kvantumkapuk és mérések rendezett sorozataként lehessen értelmezni.</span><span class="sxs-lookup"><span data-stu-id="d296f-121">Until now, to learn quantum computing you needed to learn the circuit model to understand the algorithms in the form of ordered sequences of quantum gates and measurements.</span></span> <span data-ttu-id="d296f-122">A Q# egy másik utat is járhatóvá tesz: a kvantum-számítástechnika kvantumprogramok írása révén történő elsajátítását.</span><span class="sxs-lookup"><span data-stu-id="d296f-122">With Q# you can take another path: learn quantum computing by writing quantum programs.</span></span>

## <a name="use-q-to-design-quantum-algorithms"></a><span data-ttu-id="d296f-123">A Q# használata kvantumalgoritmusok megtervezéséhez</span><span class="sxs-lookup"><span data-stu-id="d296f-123">Use Q# to design quantum algorithms</span></span>

<span data-ttu-id="d296f-124">A Q# egyre több kódtárat és felhasználó által definiált típust biztosít, amelyek segítenek az eszközök használatában és a speciális kvantumalgoritmusok megalkotásában.</span><span class="sxs-lookup"><span data-stu-id="d296f-124">Q# provides you with an increasing number of libraries and user-defined types that will help you to implement tools and build advanced quantum algorithms.</span></span> <span data-ttu-id="d296f-125">Vegyük például azt a helyzetet, hogy össze kell fonnia két qubitet (q1 és q2).</span><span class="sxs-lookup"><span data-stu-id="d296f-125">For example, you need to entangle two-qubits q1 and q2?</span></span> <span data-ttu-id="d296f-126">A qubitek összefonásához szükséges kapuk egyenkénti alkalmazása helyett alkalmazhatja a már beépített `PrepareEntangledState([q1], [q2])` műveletet.</span><span class="sxs-lookup"><span data-stu-id="d296f-126">Instead of applying individually the necessary gates to get the qubits entangled you can use the already built-in operation `PrepareEntangledState([q1], [q2])`.</span></span>

<span data-ttu-id="d296f-128">A Q#-program végrehajtásának szimulálása a teljes állapotú kvantumszimulátor használatával lehetséges, amely a Quantum Development Kit (QDK) része.</span><span class="sxs-lookup"><span data-stu-id="d296f-128">You can simulate the execution of your Q# program using the full state quantum simulator that is provided with the Quantum Development Kit (QDK).</span></span>  <span data-ttu-id="d296f-129">A QDK olyan erőforrásbecslő eszközöket is tartalmaz, amelyek megállapításokat nyújtanak az olyan Q#-programok teljesítményéről, amelyek túl nagyon a szimulátoron való futtatáshoz.</span><span class="sxs-lookup"><span data-stu-id="d296f-129">The QDK also provides resource estimators that give you insights on the performance of Q# programs that are too big to be run on a simulator.</span></span>  <span data-ttu-id="d296f-130">Ezek rendkívül értékesek az algoritmusok tervezői számára, akik ezáltal úgy tudják finomhangolni a programjaikat, hogy azok kevesebb erőforrást használjanak (például kevesebb qubit fusson kevesebb műveletre esően) és futtathatók legyenek a korai, kisebb léptékű kvantumhardvereken is.</span><span class="sxs-lookup"><span data-stu-id="d296f-130">This is highly valuable for algorithm designers, because they can tune their programs to use fewer resources (e.g. fewer number of qubits running for fewer numbers of operations), to run on earlier smaller scale quantum hardware.</span></span>

<span data-ttu-id="d296f-132">A Q# szépsége az, hogy a programokat elég egyszer megírni, majd kvantumszimulátorokon való futtatásukkal elvégezhető a hibakeresésük, valamint több kvantumszámítógép hardverén is futtathatók.</span><span class="sxs-lookup"><span data-stu-id="d296f-132">The beauty of Q# is that a program can be written once and run on quantum simulators for debugging, and run on multiple quantum computer hardware.</span></span>  <span data-ttu-id="d296f-133">Ahogy a kvantumszámítógépek fejlődnek, és új kvantumszámítógépek válnak elérhetővé, a Q# nyelven írt teljesítménytesztelési programok futtathatók a hardverteljesítmény ellenőrzése és az eredmények összehasonlítása céljából.</span><span class="sxs-lookup"><span data-stu-id="d296f-133">Benchmark programs written in Q# can be run to validate hardware performance and compare results as quantum computers evolve and new quantum computers become available.</span></span>
